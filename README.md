# 🌙 ScreenDimmer

> Proteja seus olhos. Reduza o brilho além do limite.

[![Versão](https://img.shields.io/badge/versão-1.0.0-blue.svg)](https://github.com/EduardoSpek/ScreenDimmer/releases)
[![Windows](https://img.shields.io/badge/Windows-10%2F11-00adef.svg)](https://github.com/EduardoSpek/ScreenDimmer/releases)
[![Licença](https://img.shields.io/badge/licença-MIT-green.svg)](LICENSE)

**ScreenDimmer** é um software gratuito para Windows que reduz o brilho da tela além dos limites do monitor. Ideal para quem trabalha à noite ou em ambientes com pouca luz.

![ScreenDimmer Preview](https://eduardospek.github.io/ScreenDimmer/images/preview.png)

---

## ✨ Funcionalidades

- 🎚️ **Ajuste fino de opacidade** (10% a 80%)
- 🎨 **Cores personalizáveis** (preto, sépia, azul noturno)
- 🖱️ **Click-through** - Continue usando seus programas normalmente
- 🔔 **System Tray** - Acesse presets rapidamente pelo ícone da bandeja
- 💾 **Salva configurações** - Lembra sua última configuração
- 🚀 **Inicia com Windows** - Opcional, configure conforme sua preferência

## 🚀 Como Usar

1. **Execute** o `ScreenDimmer.exe`
2. **Ajuste** a intensidade do slider (10% a 80%)
3. **Escolha** a cor do filtro (clique em "Cor")
4. **Minimize** para o System Tray se preferir
5. **Clique através** da camada escura normalmente!

### Atalhos de Teclado

- `Ctrl + Shift + ↑` - Aumentar brilho
- `Ctrl + Shift + ↓` - Diminuir brilho
- `Ctrl + Shift + D` - Mostrar/ocultar
