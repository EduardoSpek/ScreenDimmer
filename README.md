# ScreenDimmer - Landing Page

Uma landing page profissional e moderna para o aplicativo ScreenDimmer.

## 📁 Arquivos

- `index.html` - Página principal
- `styles.css` - Estilos e animações
- `script.js` - Interatividade
- `ScreenDimmer.exe` - Executável do aplicativo (adicione aqui)

## 🚀 Como Publicar

### Opção 1: GitHub Pages (Grátis e Recomendado)

1. Crie um repositório no GitHub
2. Faça upload dos arquivos
3. Vá em **Settings > Pages**
4. Selecione a branch `main` e pasta `/root`
5. Seu site estará em `https://seuusuario.github.io/screendimmer`

### Opção 2: Netlify (Grátis)

1. Acesse [netlify.com](https://netlify.com)
2. Arraste a pasta para a área de upload
3. Pronto! URL personalizada gerada automaticamente

### Opção 3: Vercel (Grátis)

1. Acesse [vercel.com](https://vercel.com)
2. Importe do GitHub ou faça upload
3. Deploy automático!
