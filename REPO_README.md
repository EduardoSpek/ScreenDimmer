# 🌙 ScreenDimmer

Um redutor de brilho elegante para Windows que protege seus olhos durante a noite.

## ✨ Recursos

- 🎨 **Ajuste de opacidade** de 10% a 80%
- 🌈 **Filtro de cores** personalizável
- 🔧 **Presets rápidos** (10%, 20%, 30%, etc.)
- 💾 **Lembra última configuração**
- 🔔 **Minimiza para System Tray**
- 🖱️ **Click-through** - clique através do filtro
- ⚡ **Inicia com Windows** (opcional)

## 📥 Download

Baixe a versão mais recente: **[ScreenDimmer_Setup.exe](https://github.com/eduardospek/screendimmer/releases/latest/download/ScreenDimmer_Setup.exe)**

Ou visite nossa landing page: **https://eduardospek.github.io/screendimmer**

## 🚀 Instalação

1. Baixe o `ScreenDimmer_Setup.exe`
2. Execute o instalador
3. Pronto! O ScreenDimmer já está no seu computador

## 🖥️ Requisitos

- Windows 10 ou 11
- 10 MB de espaço em disco
